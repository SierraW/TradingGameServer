#PTG Production JSON Guideline
######version 0.1
##Structure
```json
{
  "name": "Canadian Dollar",
  "prefix": "CAD",
  "symbol": "$"
}
```
##Explanation
### name
The name of the currency.
### prefix
The prefix of the currency, no more than 5 characters.
### symbol
The symbol of the currency, should only contain one character.
